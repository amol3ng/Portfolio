# Amohelang Mokhele — Portfolio Website

A personal portfolio website built with **Vue.js** and **Node.js**, showcasing projects, skills, certifications, and contact information.

---

## 🚀 Tech Stack

- **Frontend:** Vue.js
- **Backend:** Node.js
- **Database:** MySQL

---

## 📦 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/amol3ng/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:3002`
